JavaScript

Mit JavaScript kann man verschiedene Animationen erstellen oder Figuren designen. Wir haben auf dem Programm p5.js gearbeitet. Das Programm liest den Code, den man geschrieben hat, immer von oben nach unten. Man kann mithilfe von JavaScript auch einstellen, dass, wenn man mit der Maus aufs Bild klickt, sich etwas verändert oder eine Animation beginnt oder stoppt.

JavaScript ist eine Programmiersprache, die in Webbrowsern verwendet wird, um Webseiten interaktiv und dynamisch zu machen. Mit JavaScript können Webseiten auf Benutzeraktionen reagieren, Daten verarbeiten und Elemente auf der Seite ändern, ohne die Seite neu zu laden. Es ist eine wichtige Technologie, die dazu beiträgt, dass moderne Websites so funktionieren, wie wir es gewohnt sind.

Bei JavaScript ist es wichtig, auf Semikolons zu achten, um Codezeilen zu trennen. Verwende klare Variablennamen und beachte die Groß- und Kleinschreibung, da JavaScript dies berücksichtigt. Halte deinen Code organisiert, vermeide globale Variablen, und lerne die Fehlermeldung in der Browserkonsole zu interpretieren, um Probleme zu finden und zu beheben. Beachte auch die Sicherheit, um vor Cross-Site Scripting-Angriffen zu schützen, indem du Benutzereingaben sorgfältig validierst und säuberst.
